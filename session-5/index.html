Exercise 1: Standardizing & Merging CSV Data

Hands-on with Copilot

Scenario

You have 40 CSV files with sales information having inconsistent column names. Example: Some have "Date", others "transaction_date".

Task

Use Copilot to Clean and Merge the Data

Expected Outcome

Single CSV file containing all data with standardized column naming.

Download Sample Data

📥 Download Exercise 1 Sales_data.zip — Contains 40 sample CSV files with inconsistent column names for this exercise.

Exercise 1: Suggested Prompt

Copy and paste this prompt into Copilot Chat:

I have 40 CSV files in a folder with sales data. The files have inconsistent column naming:
- Some files have columns: "Date", "Product", "Region", "Sales", "Quantity"
- Others have: "transaction_date", "product_name", "area", "amount", "qty"
- Column names vary across files

I need to:
1. Read all 40 CSV files
2. Standardize all column names to: date, product, region, amount, quantity
3. Merge all data into a single DataFrame
4. Export the merged data to a single CSV file

Write Python code using pandas that handles this data cleaning and merging. Include error handling for missing files and inconsistent formats.
This exercise teaches participants to translate business requirements into Copilot prompts, review generated code, and iterate. It shows how Copilot augments workflow. The goal is confidence and practical experience.

Exercise 2: Building Interactive Tool

Interactive Income Tax Calculator

Objective

Build an interactive tax calculator that computes income tax liabilities according to Indian Income Tax regulations.

Expected Outcome

A fully functional web-based calculator where users input their annual income and other financial details, receiving tax liability calculations under both the old and new tax regimes for informed decision-making.

Tax Calculator Prompt

Copy and paste this prompt into Copilot Chat:

Build an interactive Indian Income Tax Calculator with real-time dual-regime comparison (Old vs New regime).

REQUIREMENTS:

1. INPUT FIELDS:
   - Gross annual income (salary/business/other sources)
   - Age (to determine applicable exemption limits)
   - HRA (House Rent Allowance) - only for salaried
   - Investment deductions (Section 80C: max ₹1,50,000) - ELSS, Life Insurance, Fixed Deposits, PPF, etc.
   - Medical/Health Insurance premium (Section 80D):
     * Self & Family: up to ₹25,000; Senior Citizens (60-80 yrs): up to ₹50,000; Super Senior (80+ yrs): up to ₹1,00,000
   - Education Loan Interest (Section 80E): no limit
   - Donations (Section 80G): approved charities and political parties
   - Home Loan Interest (self-occupied property): up to ₹2,00,000
   - Disability deductions (Section 80U): up to ₹1,25,000

2. TAX SLABS (FY 2025-26):
   OLD REGIME (standard individuals <60 years):
   - Up to ₹2.5L: 0% | ₹2.5L-₹5L: 5% | ₹5L-₹10L: 20% | Above ₹10L: 30%
   - Standard Deduction: ₹50,000 | Rebate (Sec 87A): ₹12,500 (tax-free up to ₹5L)
   
   NEW REGIME (default):
   - Up to ₹4L: 0% | ₹4L-₹8L: 5% | ₹8L-₹12L: 10% | ₹12L-₹16L: 15% | ₹16L-₹20L: 20% | ₹20L-₹24L: 25% | Above ₹24L: 30%
   - Standard Deduction: ₹75,000 | Rebate (Sec 87A): ₹60,000 (tax-free up to ₹12L)

3. DEDUCTIONS ALLOWED:
   OLD REGIME (all below available):
   - Section 80C: Investments up to ₹1,50,000
   - Section 80D: Medical insurance (caps by age)
   - Section 80E: Education loan interest (no limit)
   - Section 80G: Charitable donations (50% or 100% deduction per donation type)
   - Section 80U: Disability deduction (up to ₹1,25,000)
   - Home Loan Interest (self-occupied): up to ₹2,00,000; Let-out property: no limit
   - HRA (House Rent Allowance): standard HRA calculation
   
   NEW REGIME (NO deductions except):
   - Standard Deduction: ₹75,000 only
   - NPS employer contribution Section 80CCD(2): up to 14% of basic pay
   - Agniveer corpus fund Section 80CCH (allowed in both regimes)

4. CALCULATIONS:
   OLD REGIME FORMULA:
   Taxable Income = Gross Income - Standard Deduction (₹50K) - All applicable deductions
   Tax = Apply slab rates on Taxable Income
   Tax after Rebate = Tax - Min(₹12,500, calculated tax)
   Cess = 4% on tax amount if tax > ₹1,00,000
   
   NEW REGIME FORMULA:
   Taxable Income = Gross Income - Standard Deduction (₹75K)
   Tax = Apply slab rates on Taxable Income
   Tax after Rebate = Tax - Min(₹60,000, calculated tax)
   Cess = 4% on tax amount if tax > ₹1,00,000

5. OUTPUT DISPLAY:
   - Side-by-side comparison: Old Regime vs New Regime
   - Show breakdown: Gross Income → Deductions → Taxable Income → Tax → After Rebate → With Cess
   - Effective tax rate (%) for each regime
   - Recommendation: Which regime saves more tax and by how much
   - List deductions used in old regime and amounts claimed
   - Display unused deduction capacity

6. FEATURES:
   - Real-time calculations as user inputs change
   - Clean, professional two-column comparison layout
   - Easy regime switching/toggling
   - Mobile-responsive design
   - Input validation (non-negative amounts, max limits)
   - Age-based deduction limit auto-adjustment (60+, 80+ seniors)
   - Tooltip/help text explaining each input field
   - Save comparison results as PDF/JSON option
   - Show "breakeven deduction" point (when old regime becomes better)

REFERENCE INFO:
- Tax-free income old regime: ₹5L | New regime: ₹12L
- For ₹12L income: both regimes are comparable; above ₹12L new regime generally better unless significant deductions claimed
- Switching between regimes: possible each financial year (except certain conditions)
This exercise demonstrates building a practical, real-world tool with Copilot. Participants will see how to rapidly prototype a calculator with specific domain knowledge (Indian tax laws). This shows Copilot's ability to handle specialized requirements and iterate on complex logic.
