Setup Guide

Windows Development Environment Setup

Complete step-by-step instructions for setting up Python, VS Code, and GitHub Copilot on Windows.

1
Install Python 3.12.9 on Windows
Stable LTS-class release — widely supported by all major libraries
⭐ Recommended: Python 3.12.9
1
Download the Windows installer
Go to python.org/downloads/release/python-3129 — scroll to the Files section and click "Windows installer (64-bit)". This downloads python-3.12.9-amd64.exe.
Not sure if your Windows is 64-bit? Press Win + Pause/Break or go to Settings → System → About. Almost all modern laptops are 64-bit.
2
Run the installer — don't miss this step
Double-click the .exe. On the first screen check both options at the bottom:
☑ Use admin privileges when installing py.exe
☑ Add python.exe to PATH

Then click Install Now.
Critical: If you skip "Add python.exe to PATH", your terminal won't recognise the python command and you'll need to reinstall.
3
Disable the PATH length limit
At the end of installation, click "Disable path length limit" when prompted — this prevents errors with deeply nested project folders on Windows.
4
Verify the installation
Press Win + R, type cmd, hit Enter. Run python --version — you should see Python 3.12.9. Also run pip --version to confirm the package manager is ready.
2
Install VS Code on Windows
Your code editor
1
Download VS Code
Go to code.visualstudio.com — click "Download for Windows". This gives you VSCodeSetup-x64.exe.
2
Install with the right options
Run the installer, accept the license, and on the Select Additional Tasks screen check all of these:
☑ Add "Open with Code" to Windows Explorer file context menu
☑ Add "Open with Code" to Windows Explorer directory context menu
☑ Add to PATH
☑ Register Code as an editor for supported file types
The "Open with Code" options let you right-click any folder in File Explorer and open it directly in VS Code — very handy.
3
Install the Python extension
Open VS Code → press Ctrl + Shift + X → search Python → install the one published by Microsoft. This adds syntax highlighting, IntelliSense, and debugging.
4
Select Python 3.12.9 as your interpreter
Press Ctrl + Shift + P → type Python: Select Interpreter → select Python 3.12.9 from the list.
Don't see it? Click "Enter interpreter path" and browse to C:\Users\YourName\AppData\Local\Programs\Python\Python312\python.exe.
5
Test it with a quick file
Press Ctrl + N → save as hello.py → type print("Hello, World!") → press F5. You should see the output in the Terminal panel at the bottom.
3
Set up GitHub Copilot
Your AI coding assistant inside VS Code
👥 Team account convention — follow this for all members
Use your official Square email ID to register on GitHub. This ensures your account is linked to the organisation and is auditable by the team admin.

Your GitHub username must start with the prefix PS followed by your name or identifier. Keep it consistent and professional.
Examples: PSjohnsmith PSanitaroy PSrahulverma
1
Create your GitHub account with your Square email
Go to github.com → click Sign up. When prompted for an email, enter your Square email ID (e.g. yourname@squareinc.com). Complete the email verification step.
Do not use a personal email. Using your Square email keeps the account traceable and eligible for any organisation-level Copilot licences the team may set up in future.
2
Set your username with the "PS" prefix
During signup, enter a username that starts with PS followed by your name, e.g. PSjohnsmith or PSanitaroy. Avoid special characters unless needed for uniqueness.
Username already taken? Try PSjohn-smith or PSanitaroy2 — always keep the PS prefix.
3
Activate GitHub Copilot on your account
Go to github.com/features/copilot → click Get Copilot Free. No credit card is required for the free plan.
Student? Go to education.github.com/pack and verify your student status to get Copilot Pro for free.
4
Install GitHub Copilot extensions in VS Code
Press Ctrl + Shift + X → search GitHub Copilot → install both:
• GitHub Copilot — inline code suggestions
• GitHub Copilot Chat — AI chat sidebar
5
Sign in to GitHub inside VS Code
A pop-up appears in the bottom-right — click Sign in to GitHub. Your browser opens → click Authorize Visual-Studio-Code → allow when Windows asks → you'll be redirected back to VS Code automatically.
No pop-up? Click the Accounts icon (person icon) in the bottom-left of VS Code → Sign in with GitHub to use GitHub Copilot.
⚠️ Important — free plan usage limits
The free version of GitHub Copilot comes with limited monthly usage that resets each month. Once the quota is exhausted, Copilot will stop working until the next reset.
2,000 code completions / month
50 chat messages / month

Please do not exhaust your quota before the Friday workshop. Avoid casually experimenting with Copilot in the days leading up to it — save your usage so you have the full experience available during the session. The workshop is where you'll get the most value from it.
